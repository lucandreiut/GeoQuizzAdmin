# GeoQuizzAdmin

``` bash



# Move into /DockerGeoQuizz/Postgres
# Run :
docker build -t lpro/pg11 .

# Move into /Docker/Wildfly admin
# Run :
docker build -t lpro/wildfly-admin .

# Move back to /DockerGeoQuizz
# Run :
docker-compose up
```
